Kolibri Style Overview

Kolibri comes with its default set of colors, shadows, fonts, and other style elements. Any application that uses Kolibri can of course selectively override or totally ignore the default style.

The style is available in two ways: via kolibri-base.css and by using the style API as provided by the style module . Sometimes, you might even want to combine both approaches, like the live test report does.
